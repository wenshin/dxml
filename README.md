# DXML

DXML is designed for more effective to change UI design into code, even automatic.

# Install

```bash
npm install dxml
yarn install dxml
```

# Features

see [docs](https://wenshin.github.io/projects/dxml/)

# Development

## Available Scripts

In the project directory, you can run:

`yarn start`

### DXML package Build

`yarn build-pkg`

### Doc build

`yarn build`
